##硬件改装
 - 思路: 1u 电源的风扇噪音太大，另外机箱风扇的质量也不怎么样，先把电源全部更换了，再看看机箱风扇的效果；如果机箱风尚不给力，则把机箱风扇给更换了。
 - 电源功耗分析: 根据网络上对 j3455 黑群晖功耗的实测，实际不会超过 60w， 所以按照 60w的功耗标准来进行电源采购即可。
